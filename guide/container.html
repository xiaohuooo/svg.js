此外，link 元素具有创建属性的方法：target()target

元素也可以与该方法相反的方式链接：linkTo()

js

或者，可以传递一个块而不是 URL，以获取链接元素上的更多选项：

js
rect.linkTo(function(link) {
  link.to('http://svgdotjs.github.io/').target('_blank')
})
